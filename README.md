## HS系列

Vue, Angular, React

## HS阶段

### 第一阶段: 基于Angular开发

* hs-ui        : 通用UI框架
* hs-ngx       : Angular的基础组件
* hs-ngx-bs    : Angular的业务组件
* hs-ngx-apps  : Angular的多应用基础框架

### 第二阶段: 基于Vue开发(be continued)

* hs-vx        : Vue的基础组件
* hs-vx-bs     : Vue的业务组件
* hs-vx-apps   : Vue的多应用基础框架

### 第三阶段: 基于React开发(be continued)

* hs-rx        : React的基础组件
* hs-rx-bs     : React的业务组件
* hs-rx-apps   : React的多应用基础框架
